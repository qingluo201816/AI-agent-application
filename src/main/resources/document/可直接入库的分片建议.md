# 文档 5：可直接入库的分片建议

## 分片维度建议
1. 按人物分片：每个人物一条主文档 + 一条隐藏设定
2. 按世界设定分片：修炼体系、势力、地图、禁忌、资源分别独立
3. 按事件分片：每个重大事件单独一条，包含起因、经过、后果、相关人物
4. 按情节模板分片：方便在写作时快速召回“类似结构”

## 推荐元数据字段
- `doc_type`：character / world / event / plot_rule / faction / location
- `name`：文档名称
- `era`：上古 / 千年前 / 当前时代
- `importance`：high / medium / low
- `visibility`：public / hidden
- `related_entities`：相关人物或势力名
- `keywords`：标签关键词

## 示例 JSON 结构
```json
{
  "doc_type": "character",
  "name": "林渊",
  "visibility": "public",
  "keywords": ["主角", "隐忍", "灭门案", "界源碎印"],
  "content": "林渊，17岁，边陲小城没落家族后人，性格沉静寡言，观察细致，核心目标是查清林家灭门真相。"
}
```
