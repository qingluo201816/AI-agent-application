# 文档 2：人物档案库

> 建议：每个核心人物单独成条，包含“公开信息”和“隐藏信息”，方便 RAG 在生成剧情时控制信息暴露层次。

## 2.1 主角档案
- **姓名**：林渊
- **年龄**：17
- **身份**：边陲小城没落家族后人
- **初始形象**：沉静寡言、观察细致、外冷内热
- **性格标签**：隐忍、谨慎、重情、强自尊、对背叛极度敏感
- **核心欲望**：查清家族灭门真相，保护仅存亲人，摆脱任人摆布的命运
- **成长弧线**：从被动求生到主动布局，从只顾身边人到承担苍生因果
- **能力特点**：悟性高，神识异常敏锐，对古碑、残魂、封印类传承有天然共鸣
- **弱点**：不擅主动表达情感，容易把责任全部压在自己身上
- **隐藏设定**：体内沉睡着一枚界源碎印，会逐步影响他的气运、敌人感知与命运轨迹

## 2.2 女主 / 核心搭档
- **姓名**：苏晚照
- **年龄**：18
- **身份**：丹霞谷真传弟子
- **外在气质**：清冷理智，行事克制
- **性格标签**：聪明、审慎、嘴硬心软、原则感强
- **核心功能**：治疗、智囊、情绪平衡器、推动主角学会信任
- **能力特点**：丹道、毒理、灵识辨析极强，擅长发现异常线索
- **人物矛盾**：她表面属于中立宗门，实则背负家族旧案，不愿卷入大势，却注定无法置身事外
- **与主角关系演化**：互相利用 → 互相信任 → 并肩承担 → 感情升温但被阵营矛盾阻隔

## 2.3 宿敌角色
- **姓名**：顾承霄
- **年龄**：20
- **身份**：太一玄宗核心弟子
- **性格标签**：骄傲、自律、极度理性、胜负心强
- **人物定位**：主角前期压迫者，中期对照组，后期复杂盟友
- **信念**：秩序高于一切，个人情感不能凌驾大局
- **能力特点**：雷法、阵道双修，战斗风格堂皇强势
- **内在矛盾**：他并非纯粹反派，而是相信只有稳定秩序才能避免世界再次崩坏

## 2.4 妖族角色
- **姓名**：白璃
- **身份**：万妖山狐族少主
- **性格标签**：慵懒、狡黠、敏锐、善于试探人心
- **作用**：推动妖族线、情报线和阵营灰度冲突
- **人物魅力点**：说话轻佻，但对族群命运极认真
- **隐藏设定**：她知道一部分上古封天之战的真相，但无法全部说出

## 2.5 师长角色
- **姓名**：沈孤舟
- **身份**：青冥剑阁弃徒 / 散修强者
- **人物定位**：主角重要导师
- **性格标签**：冷硬、寡言、极护短、看似不近人情
- **功能**：提供战斗训练、价值观冲击、旧时代视角
- **人物秘密**：当年参与过某场禁忌之战，知道主角家族被灭并非表面那么简单

## 2.6 反派核心角色
- **姓名**：祁夜
- **身份**：冥河教圣子
- **性格标签**：温和、优雅、病态、极端坚定
- **核心追求**：打破现世秩序，迎回“真正的神明时代”
- **危险点**：他相信自己是在拯救世界，因此不认为自己邪恶
- **与主角关系**：既是敌人，也是“真相引路人”

## 2.7 人物关系网简表
- 林渊 ↔ 苏晚照：信任逐步建立，情感缓慢升温
- 林渊 ↔ 顾承霄：竞争、误解、彼此认可
- 林渊 ↔ 白璃：合作与试探并存
- 林渊 ↔ 沈孤舟：师徒名分浅，但精神影响极深
- 林渊 ↔ 祁夜：立场对立，但都在追寻世界真相
