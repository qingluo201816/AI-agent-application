## 作品总览

### 作品概述
- entry_type: overview

《斗破苍穹》是一部以“斗气大陆”为舞台的长篇玄幻成长小说，核心叙事围绕萧炎从“天才陨落”到“炎帝崛起”的全过程展开。故事主线同时包含成长、复仇、师徒、家族、宗门博弈、异火争夺、远古种族之争以及终极大战等多层线索。

全书的叙事骨架非常清晰：前期在乌坦城与加玛帝国完成个人命运转折，中期借迦南学院、黑角域与中州舞台完成实力跃迁，后期则将冲突扩展为魂族、古族等远古势力层面的大陆级浩劫。人物成长与世界设定推进是同步进行的。

### 核心主题
- entry_type: overview
- aliases: 成长主题|逆袭主题
- related_people: 萧炎|纳兰嫣然|药尘|魂天帝
- importance: high

本作最鲜明的主题是“逆境成长”和“尊严夺回”。萧炎在天赋消失、家族冷眼、婚约羞辱等多重压迫下重建自我，其名句“三十年河东，三十年河西，莫欺少年穷”成为整部作品的精神坐标。

此外，小说还反复书写“传承”的意义：药老对萧炎的传承、萧族血脉对后人的召唤、陀舍古帝传承对时代格局的决定，都是把个人奋斗与历史余脉相连接的重要装置。

## 世界观与规则

### 斗气大陆
- entry_type: world_rule
- aliases: 大陆|斗气世界
- related_people: 萧炎
- importance: core

斗气大陆是一个以斗气为绝对主流力量体系的世界。这里没有传统西幻中的魔法主导秩序，修炼、战斗、宗派组织、资源争夺与社会层级几乎都围绕斗气展开。大陆地域辽阔，存在帝国、学院、宗门、佣兵体系、魔兽族群与远古种族等多重力量结构。

大陆前中期叙事主要落在加玛帝国、黑角域、迦南学院等区域，后期则逐步扩展到中州与远古种族活动范围，最终上升到影响整个大陆命运的终局战争。

### 强弱判定规则
- entry_type: world_rule
- aliases: 战力判定|强者标准
- importance: high

本作中强弱并不只由境界决定。通常需要综合三类因素判断：第一是修为境界本身；第二是修炼功法等级；第三是斗技、异火、灵魂力量、战斗经验、体质与临场手段等附加条件。

这意味着“低境界凭手段越级而战”在本作中是合理的，尤其对萧炎这样兼具高灵魂强度、异火、炼药术和复合斗技的人物而言，更是如此。

### 源气与斗帝断层
- entry_type: world_rule
- aliases: 斗帝断代|源气
- related_people: 陀舍古帝|魂天帝|萧炎
- importance: core

斗帝并非只是更高一级的普通境界，而是与天地根源之力“源气”相关。由于后世大陆失去源气，斗帝道路长期断绝，导致斗圣巅峰已成为常规意义上的顶点。远古斗帝时代逐渐成为传说，陀舍古帝洞府与帝品雏丹因而具备决定时代命运的重量。

这一设定解释了为什么后期所有超级势力都围绕古玉、洞府、帝品雏丹与古帝传承展开生死争夺。

## 境界及战力

### 斗气境界
- entry_type: power_system
- aliases: 修炼境界|斗气等级
- importance: core

基础斗气修炼从斗之气开始，后续依次为：斗者、斗师、大斗师、斗灵、斗王、斗皇、斗宗、斗尊、半圣、斗圣、斗帝。早期的斗之气通常以段位呈现，是家族少年培养与测试的基础；真正凝聚斗气旋后，才算正式迈入斗者体系。

小说前中后期的篇章升级非常明显：加玛帝国阶段以斗王、斗皇为高端战力，中州阶段斗宗、斗尊成为主流门槛，最终则进入斗圣与斗帝层级的终极碰撞。

### 功法等级
- entry_type: power_system
- aliases: 天地玄黄|修炼功法
- importance: high

功法分天、地、玄、黄四阶，每阶又可细分初中高三级。功法等级决定修炼效率、斗气质量、上限潜力与持续作战能力，是长期成长曲线的重要基础。

萧炎前期最关键的机缘之一，就是得到药老传授、可通过吞噬异火持续进化的《焚诀》。《焚诀》打破了固定品阶功法的常规逻辑，是萧炎后期能够一路冲顶的根本支柱之一。

### 斗技等级
- entry_type: power_system
- aliases: 战技|秘法
- importance: high

斗技同样分天、地、玄、黄四阶，用于具体战斗释放。不同斗技既有正面攻伐类型，也有身法、封印、辅助、灵魂层面的特殊用途。高等级斗技往往掌握在宗门、学院、远古种族或强者传承中。

萧炎在不同阶段积累了多种标志性斗技和秘法，如爆步、吸掌、八极崩、焰分噬浪尺、佛怒火莲等，它们共同构成了他的战斗风格。

### 炼药师体系
- entry_type: power_system
- aliases: 炼药术|丹药体系
- related_people: 药尘|古河|萧炎
- importance: core

炼药师在斗气大陆地位极高，因为其能够炼制各种恢复、突破、救命、辅助、塑体类丹药。成为炼药师通常要求火属性体质中兼具木属性、并拥有优秀灵魂感知力，因此门槛远高于普通修炼者。

炼药师体系对本作意义极大：它不只是职业设定，也是资源与人脉网络的核心。萧炎借药老的传承踏入炼药领域后，不仅提升战斗续航和复活手段，更以炼药术在中州建立名望，成为凝聚联盟的重要资本。

### 灵魂境界
- entry_type: power_system
- aliases: 灵魂力量|灵魂感知
- related_people: 萧炎|药尘
- importance: high

灵魂力量在炼药、控火、感知、布置灵魂攻击与操控高级手段时极为关键。萧炎因两世灵魂与长期炼药修炼，灵魂基础显著强于常人，这也是他早期被视为修炼天才的重要原因之一。

到了后期，灵魂境界更成为斗圣层面决定胜负的关键因素之一，尤其在争夺高阶异火、炼制高阶丹药与对抗魂族时作用突出。

### 异火体系
- entry_type: power_system
- aliases: 天地异火|异火榜
- related_people: 萧炎|药尘|净莲妖圣|虚无吞炎
- importance: core

异火是天地间诞生的顶级火焰，拥有远胜普通兽火与斗气火焰的威能，既可用于炼药，也可用于战斗与特殊进化。异火之间存在榜单与强弱差异，不同异火具备不同特性。

萧炎之所以能从同辈中脱颖而出，很大程度上依赖其围绕异火展开的完整成长路径：收服、炼化、融合、以焚诀进化，最终走向帝炎与斗帝之位。异火也是全书最核心的资源竞争轴之一。

### 萧炎
- entry_type: character

萧炎是全书绝对主角，出身乌坦城萧家，早年曾是举族瞩目的修炼天才，却因药老灵魂寄宿在戒指中持续吸收斗气而突然沦为“废物”。这一跌落让他尝尽人情冷暖，也构成其全部逆袭叙事的起点。

在药老现身后，萧炎获得真正意义上的修炼与炼药传承，先后经历魔兽山脉历练、沙漠异火争夺、三年之约、黑角域博弈、迦南学院成长、中州扬名、远古种族争锋与大陆终局大战，最终依靠焚诀、异火融合、灵魂力量与古帝传承踏入斗帝层次，成为“炎帝”。

### 药尘

药尘给予萧炎的不只是功法、斗技、炼药术和异火认知，更重要的是战略眼界与强者气度。二人的师徒关系贯穿始终，是作品中最稳固也最有情感厚度的关系之一。后期萧炎又反过来为药老夺骨、炼体、复活，形成完整的传承闭环。

薰儿在情感线上代表“始终如一”的守候，在主线层面则承担着把萧炎从加玛帝国世界引向远古种族格局的桥梁作用。她与萧炎的感情并非简单青梅竹马，而是建立在共同成长、长期信任和并肩承担命运之上的关系。

随着剧情推进，彩鳞逐渐从“危险且不可控的女王”转变为萧炎阵营中极其重要的高端战力和情感核心之一。她的存在让萧炎与蛇人族、沙漠势力和后期更大范围的联盟关系获得稳定支点。

她的条线把“体质天赋既是馈赠也是诅咒”这一主题表现得很鲜明。萧炎对她的帮助既是情义，也体现出主角从少年强者向真正能够承担他人命运的人物转变。

### 魂天帝

云岚宗是加玛帝国顶级宗门，也是前期最强大的宗派力量之一。它既与纳兰嫣然退婚事件深度绑定，也是三年之约和后续家仇冲突的核心舞台。

云岚宗在前中期是萧炎必须翻越的庞然大物，后期则因云山、魂殿渗透与宗门失控而走向覆灭，成为“盛极而衰”的典型势力样本。

## 功法、异火与重要物件

### 焚诀
- entry_type: artifact_or_technique
- aliases: 焚决
- related_people: 萧炎|药尘|陀舍古帝
- importance: core

《焚诀》是萧炎最重要的功法，其最大特点在于可通过吞噬异火不断进化。它并非普通意义上的固定等级功法，而是一条能伴随主角一起成长的特殊道路。

从结构上说，焚诀几乎决定了全书的资源路线：异火为什么必须争、为什么争到手后能产生远超单次机缘的价值，都与此直接相关。

- entry_type: artifact_or_technique
- aliases: 火莲
- related_people: 萧炎
- importance: core

佛怒火莲是萧炎最具标志性的招牌斗技之一，其核心在于将不同异火进行高危险度融合后释放出极端爆发力。它代表了萧炎“敢赌、敢拼、敢把高风险机缘转成战斗答案”的风格。

## 主要情节

### 天才陨落与废物时期
- entry_type: plot_event
- aliases: 开篇主线
- related_people: 萧炎|萧战|萧薰儿
- importance: core

故事从萧炎在家族测试中只剩三段斗之气写起，曾经名震乌坦城的天才彻底跌落神坛。围绕这一下坠，小说快速建立了外部羞辱、内部孤独与父女兄妹式温情支撑的三重氛围。

这一阶段的功能，是把主角放到最低位置，让后续一切成长都具备可感的反差与情绪回报。

### 退婚与三年之约
- entry_type: plot_event
- aliases: 悔婚线
- related_people: 萧炎|纳兰嫣然|云岚宗|萧战
- importance: core

纳兰嫣然借云岚宗之势登门退婚，是整部作品最具标志性的开篇高潮之一。萧炎在极端屈辱中立下三年之约，成为其最初且最明确的阶段目标。

这一事件彻底把个人耻辱转化为成长引擎，也让云岚宗成为主角前期必须翻越的最大障碍。

### 药老现身与正式修行
- entry_type: plot_event
- aliases: 拜师线
- related_people: 萧炎|药尘
- importance: core

药老身份揭晓后，萧炎终于得知自己天赋陨落的真正原因。与此同时，他也第一次拥有了超越家族层级的修炼眼界、炼药传承与异火道路。

这既是主角修行体系真正启动的节点，也是全书师徒线正式成形的起点。

### 魔兽山脉历练
- entry_type: plot_event
- aliases: 山脉历练
- related_people: 萧炎|小医仙|云韵
- importance: high

魔兽山脉是萧炎第一次大规模脱离舒适区的成长阶段。这里既有生死磨砺，也让他结识小医仙、遭遇云韵，使后续情感与支线格局被提前埋下。

### 沙漠异火与美杜莎线
- entry_type: plot_event
- aliases: 青莲地心火篇
- related_people: 萧炎|彩鳞
- importance: core

萧炎进入塔戈尔大沙漠争夺青莲地心火，与蛇人族和美杜莎女王发生深度碰撞。这一篇章是主角第一次真正意义上与大陆级资源、顶级强者和危险机缘正面搏杀。

它既推动了焚诀的进化，也把彩鳞推入主线核心关系网。

### 三年之约与云岚山之战
- entry_type: plot_event
- aliases: 云岚宗前期决战
- related_people: 萧炎|纳兰嫣然|云岚宗|云韵
- importance: core

三年后，萧炎兑现约定登上云岚山，与纳兰嫣然完成公开对决。这场胜负不只是年轻一辈较量，更是对早年耻辱的正面回应。

而在这场决斗之后，萧炎与云岚宗、云山、云韵之间的矛盾迅速升级，故事也由私人恩怨转向宗门层面的全面冲突。

### 黑角域、迦南学院与陨落心炎
- entry_type: plot_event
- aliases: 学院成长篇
- related_people: 萧炎|韩枫|紫妍
- importance: core

进入迦南学院后，萧炎不仅获得了更系统的成长环境，还开始建立自己的组织能力与同辈影响力。围绕天焚炼气塔与陨落心炎展开的剧情，是其中最关键的部分。

这一阶段同时把韩枫、黑角域与药老旧怨逐渐拉进主线，使萧炎的战斗不再只是为自己，而是为师门与更大格局而战。

### 韩枫之死与药老旧仇清算
- entry_type: plot_event
- aliases: 师仇线
- related_people: 萧炎|韩枫|药尘
- importance: high

韩枫作为药老叛徒，其败亡具备极强的情绪释放价值。萧炎替师门复仇，也是其从“受教者”向“真正继承者”转变的关键一步。

### 中州崛起与救药老
- entry_type: plot_event
- aliases: 中州篇
- related_people: 萧炎|药尘|魂殿|风尊者
- importance: core

来到中州后，萧炎面对的是更成熟也更残酷的强者体系。药老、星陨阁、魂殿、丹塔、远古种族等多方势力交织，使故事层次急剧抬升。

救药老与重建药老一系力量，是这一阶段最重要的主线之一。

### 丹会与三千焱炎火
- entry_type: plot_event
- aliases: 丹塔篇
- related_people: 萧炎|丹塔

丹会让萧炎在“强者”之外，又获得“顶级炼药师新星”的名望。炼药层面的认可极大拓宽了他的资源获取与盟友吸引能力，而三千焱炎火则进一步夯实了其异火体系。

### 古界、天墓与血脉传承
- entry_type: plot_event
- aliases: 远古种族篇
- related_people: 萧炎|萧薰儿|萧玄|古元
- importance: core

这一阶段标志着萧炎正式踏入远古八族的世界。天墓中与萧玄相遇并获得血脉传承，是他后期实力爆发式跃迁的决定性转折。

### 净莲妖火争夺
- entry_type: plot_event
- aliases: 妖火篇
- related_people: 萧炎|魂族|云韵|紫妍

净莲妖火篇密度极高，集合了异火争夺、顶级强者对撞、远古机缘解谜与多方势力角力，是终局大战前最重要的资源争夺战之一。

### 古帝洞府与终局大战
- entry_type: plot_event
- aliases: 成帝之争|浩劫
- related_people: 萧炎|魂天帝|古元|虚无吞炎
- importance: core

围绕古玉、古帝洞府、帝品雏丹与古帝传承，所有主要矛盾最终汇流到一处。魂天帝冲击斗帝，萧炎则以古帝传承与异火整合对抗时代浩劫。

这不仅是主角的个人终点，也是整个斗气大陆的时代拐点。

### 炎帝诞生
- entry_type: plot_event

- related_people: 萧炎|魂天帝
- importance: core

最终，萧炎踏入斗帝层次，以炎帝之名完成对魂天帝与大陆浩劫的终结。至此，“废物少年”与“炎帝”之间形成完整闭环，整部作品的主题在此落地。

## 核心人物详细时间线

### 萧炎详细生平时间线
- entry_type: character_timeline
- aliases: 萧炎成长线|炎帝成长史
- related_people: 药尘|萧薰儿|彩鳞|魂天帝
- importance: core

1. 少年天才阶段：在乌坦城萧家表现出极高修炼天赋，是家族与外界公认的希望。
2. 天赋陨落阶段：因药老灵魂寄宿吸收斗气，连续多年停滞，遭受家族内部冷眼与外部羞辱。
3. 退婚与立誓阶段：纳兰嫣然上门退婚，萧炎在巨大屈辱中立下三年之约，人生主线就此固定。
4. 拜师重修阶段：药老现身，萧炎系统学习焚诀、炼药术与实战手段，正式走上逆袭路线。
5. 沙漠与异火阶段：在塔戈尔大沙漠争夺青莲地心火，并与美杜莎、海波东等关键人物建立复杂联系。
6. 云岚宗清算阶段：兑现三年之约，与云岚宗和云山冲突彻底激化，从帝国层面成为焦点人物。
7. 学院与中州崛起阶段：在迦南学院建立磐门，后进入中州，以炼药与战斗双线同步扬名。
8. 远古种族阶段：通过古界、天墓、萧玄传承等节点，完成血脉与层次上的真正飞跃。
9. 终局救世阶段：整合天府联盟与多方盟友，争夺净莲妖火、古帝传承，最终成帝并击败魂天帝。

### 药尘详细生平时间线
- entry_type: character_timeline
- aliases: 药老人生线|药尊者时间线
- related_people: 萧炎|韩枫|风尊者
- importance: core

1. 巅峰时期：药尘曾是大陆最顶尖的炼药师之一，兼具强大实力与炼药名望。
2. 收徒与埋祸阶段：其昔日弟子韩枫因私欲与野心，最终成为其最大背叛来源。
3. 身死残魂阶段：药尘肉身被毁，仅余灵魂寄宿戒指，长期隐藏于世。
4. 遇见萧炎阶段：在观察萧炎心性后决定现身收徒，把自己的功法、眼界与炼药术悉数传下。
5. 传承深化阶段：伴随萧炎一路成长，不仅提供战力支持，也不断校正其强者道路与行事原则。
6. 旧仇回收阶段：围绕韩枫、魂殿与自身旧部，药尘过去的历史逐步被主线回收。
7. 复活与回归阶段：在萧炎帮助下重塑肉身、恢复巅峰，完成“师传弟、弟救师”的闭环。

### 萧薰儿详细生平时间线
- entry_type: character_timeline
- aliases: 薰儿感情线|古薰儿时间线
- related_people: 萧炎|古元|凌影
- importance: core

1. 寄住萧家阶段：作为古族核心人物，被安排在萧家生活，与萧炎建立最早且最深的信任关系。
2. 守候阶段：在萧炎天赋跌落、众人轻视时依旧保持支持，是其低谷期最稳定的情感支点。
3. 暂别阶段：随着身份、家族与更大格局逐步暴露，薰儿不得不返回古族体系。
4. 再会阶段：她既是萧炎感情线的重要对象，也是把主角带入远古八族核心舞台的桥梁。
5. 并肩阶段：在古界、天墓、妖火之争与终局大战中，薰儿逐渐从“守候者”变为真正的高端战力与政治盟友。
6. 终局阶段：萧炎成长为炎帝后，她与萧炎的关系完成从青梅竹马到并肩伴侣的全面闭环。

2. 异火纠缠阶段：在围绕青莲地心火的争夺中与萧炎发生激烈交集，命运轨迹开始相互嵌合。
3. 吞天蟒过渡阶段：其状态变化与灵魂、血脉、进化问题使人物关系从单纯敌对转向复杂绑定。
4. 阵营融合阶段：随着萧炎势力增强，彩鳞逐步成为主角阵营核心高端战力之一。
5. 家庭化阶段：萧潇的存在使她与萧炎的关系不再只是激情与对抗，而带有更稳定的家庭结构意味。
6. 大战阶段：在后期联盟与终局决战中，彩鳞始终是萧炎最可靠的外围支柱之一。

5. 联军阶段：在古帝洞府与终局大战前后，紫妍及其背后的太虚古龙一族成为萧炎重要的战略依靠。

- related_people: 萧炎

## 异火详解（主线强相关）

### 帝炎
- entry_type: artifact_or_technique
- aliases: 排名第一异火|终极融合异火
- rank: 1
- related_people: 萧炎|陀舍古帝
- importance: core

帝炎并非前期即可独立争夺的常规异火，而是站在整部作品终局层面的“火之极致”象征。它代表的不只是火焰本身，更是异火体系的终极整合结果。

对萧炎而言，帝炎意味着其异火道路走到尽头，也意味着焚诀与异火融合逻辑在终章得到最终兑现。

### 虚无吞炎
- entry_type: artifact_or_technique
- aliases: 排名第二异火|魂族圣火
- rank: 2
- related_people: 魂天帝|魂族
- importance: core

虚无吞炎是魂族最关键的底牌之一，兼具诡秘、吞噬与极高战略价值。它与魂族的野心深度绑定，是魂族能在暗中长期积累优势的重要原因之一。

其用途不仅在于直接战斗，更在于辅助魂族完成更大规模的谋划与终局冲帝布局。

### 净莲妖火
- entry_type: artifact_or_technique
- aliases: 排名第三异火|妖火
- rank: 3
- related_people: 萧炎|净莲妖圣
- importance: core

净莲妖火是后期最重要的异火争夺对象之一，力量极端狂暴，且与净莲妖圣遗留历史深度缠绕。萧炎最终面对的不是简单的“收火”，而是如何在妖火灵性、反噬风险与净化条件之间找到平衡。

其融合用途在于大幅抬高萧炎异火体系的上限，也是终局成帝前的关键战力拼图。

### 金帝焚天炎
- entry_type: artifact_or_technique
- aliases: 排名第四异火|古族传承异火
- rank: 4
- related_people: 萧薰儿|古族
- importance: high

金帝焚天炎与古族密切相关，也是薰儿强大战力与高贵出身的重要象征。它的意义不仅在于威力，还在于证明古族在火焰、血脉与传承资源上的深厚底蕴。

在 RAG 问答中，金帝焚天炎应同时与“薰儿”“古族”“远古血脉”建立检索映射。

### 生灵之焱
- entry_type: artifact_or_technique

- rank: 5
- related_people: 神农老人|萧炎
- importance: high

生灵之焱的独特之处在于偏向生机、滋养和炼药辅助价值，而不只是单纯毁灭。它在高阶炼药与恢复层面的象征意义很强。

对后期的萧炎而言，这类异火价值在于进一步提升焚诀兼容性，并完善其炼药与战斗的双重体系。

### 骨灵冷火

骨灵冷火兼具森冷与炽烈两种观感，是药老最具标志性的异火之一。它最初作为药老身份的重要标记存在，也使萧炎对异火体系有了最直观的认识。

其剧情用途主要体现在师徒传承、炼药协助与异火认知启蒙层面。

### 九龙雷罡火

- rank: 12
- related_people: 焚炎谷|萧炎
- importance: medium

九龙雷罡火在书中体现的是高阶火焰资源对宗门底蕴的加成作用。它不如主角手中几种异火那样贯穿主线，但能够丰富异火体系的层次感。

在知识库中，它适合作为“中高阶异火、宗门底牌、火属性战力来源”的补充条目。

### 龟灵地火

- related_people: 药尘|萧炎

龟灵地火与炼药、稳健控火等语境的关联度较高。虽然它不像前列异火那样具备压倒性戏剧中心性，但在“药老体系”和“多异火整合”问答中有索引价值。

其条目适合被用于补足异火榜的纵深，而不是承担主线焦点。

### 陨落心炎

陨落心炎与迦南学院、天焚炼气塔和内院动荡紧密相连，是萧炎中期成长的决定性火种之一。它不仅提升战力，也深度介入炼体、修炼效率与学院篇高潮。 

对萧炎而言，其获取与融合意味着“学院成长线”和“异火成长线”的真正合流。

海心焰与韩枫关系极深，因此它在剧情里的意义远超过一簇普通异火。它不仅代表韩枫的实力，也在象征层面对应“药老传承被歪曲后的分支”。

当韩枫败亡后，海心焰相关问答通常会与“师门旧怨”“清算”“异火转移”绑定出现。

- rank: 9
- related_people: 萧炎|丹塔
- importance: core

三千焱炎火最大的特点之一是不死修复能力，这使它在实战中拥有极高的战略价值。萧炎在丹会与丹塔相关剧情后获得它，整体战斗续航因此得到跨越式提升。

其融合用途是为焚诀继续进化提供更强燃料，同时显著增强主角在高强度对轰中的生存能力。

### 青莲地心火
- entry_type: artifact_or_technique
- aliases: 排名第十九异火|青火
- rank: 19
- related_people: 萧炎|美杜莎女王|海波东

青莲地心火是萧炎真正意义上收服的第一种关键异火，也是其异火道路的起点。围绕它展开的沙漠篇把异火争夺、蛇人族、美杜莎与主角的命运正式缠绕在一起。

其融合用途主要体现在焚诀进阶、战斗爆发与主角异火路线开张三个层面。

## 远古八族

### 萧族
- entry_type: faction
- aliases: 萧家远古本族|萧玄一脉
- related_people: 萧炎|萧玄|萧战
- importance: core

萧族昔日属于远古八族之一，但在漫长岁月中早已衰败，萧炎所在的乌坦城萧家只是其残留支脉。正因为这种“曾经辉煌、如今沉寂”的反差，萧炎的成长天然带有复族意味。

萧族在后期的重要性集中体现在陀舍古帝玉、萧玄遗泽以及萧炎继承其历史责任这三条线上。

### 古族
- entry_type: faction
- aliases: 薰儿母族|远古强族
- related_people: 萧薰儿|古元|凌影
- importance: core

古族是远古八族中最稳定、最强势的超级种族之一，拥有深厚血脉、完整传承和强大高端战力。薰儿的真实身份让古族很早就与主角命运绑定。

古族在主线中的功能，既是萧炎未来的潜在盟友，也是逼迫他必须不断变强的现实标尺。

### 魂族
- entry_type: faction
- aliases: 终极反派阵营|魂殿母体
- related_people: 魂天帝|虚无吞炎
- importance: core

魂族是全书后期最核心的敌对超级势力，擅长潜伏、吞噬、收集灵魂与长线布局。魂殿可以视为其向外延展的执行体系与外壳之一。

魂族的可怕之处不只在高端战力，而在于其隐忍、耐心和对终极目标的极端执念，最终直接推动了大陆级浩劫。

### 炎族
- entry_type: faction
- aliases: 火系远古强族
- related_people: 炎烬|萧炎
- importance: high

炎族在远古八族中与火焰、异火和高爆发战力联系最紧。它与萧炎之间天然存在较高的议题交集，因此后期联盟层面有较强合作空间。

在知识库问答中，炎族通常与“异火资源”“联军构成”“远古种族对魂族态度”一起出现。

### 雷族
- entry_type: faction
- aliases: 雷霆血脉强族
- related_people: 雷赢
- importance: high

雷族同样属于远古八族中的强大战斗族群，作风偏强势，拥有相当直接的正面作战能力。其与炎族一起，是后期对抗魂族的重要明面力量。

其条目适合与“联军组成”“远古八族格局”“对魂族战争立场”做联合召回。

### 药族
- entry_type: faction
- aliases: 炼药远古强族
- related_people: 药丹|药尘
- importance: high

药族在远古八族中最具职业辨识度，与炼药、丹会、药道传承关系极深。它的重要性不只在战力，也在资源组织与高阶丹药生产能力。

从 RAG 角度看，药族条目可以与“炼药师体系”“药老身世背景”“丹塔/丹会”形成强关联。

### 石族
- entry_type: faction
- aliases: 远古强族之一
- related_people: 远古八族
- importance: medium

石族在正文中的戏份相对少于古族、魂族、炎族和雷族，但其存在本身证明了远古八族并非简单双雄对峙，而是一个曾经完整而如今不断崩落的帝族系统。

在知识库里，石族更适合作为世界观补全条目，用于回答“八族有哪些”“后期哪些种族遭魂族清洗”等问题。

### 灵族
- entry_type: faction
- aliases: 远古强族之一
- related_people: 远古八族|魂族
- importance: medium

灵族和石族一样，更多承担世界观与后期危机扩散的证明作用。其命运能侧面体现魂族行动已经从暗中布局进入对远古格局的公开重构阶段。

在问答中，灵族适合被召回到“魂族前期吞并哪些目标”“远古八族为何人人自危”等主题下。

## 重要战役（结构化）

### 三年之约决战
- entry_type: battle
- aliases: 萧炎对纳兰嫣然|云岚宗约战
- related_people: 萧炎|纳兰嫣然|云韵
- importance: core

参战方：萧炎 vs 纳兰嫣然。
胜负：萧炎完成正面取胜，兑现三年前立下的尊严之约。
代价：个人恩怨被彻底公开化，云岚宗层面的敌意被正式点燃。
后果：萧炎完成前期叙事的第一次精神复仇，但真正更大的敌人也由此现身。

### 云岚宗大战
- entry_type: battle
- aliases: 云岚山大战|云山冲突
- related_people: 萧炎|药尘|云山|云韵
- importance: core

参战方：萧炎一方 vs 云岚宗及其背后力量。
胜负：表面上并非简单碾压式完胜，但萧炎完成强势脱身并重创旧秩序威信。
代价：双方彻底结仇，萧炎不得不离开加玛帝国旧环境，进入更大舞台。
后果：云岚宗从高高在上的修炼圣地跌入不可逆的衰落轨道，主角也转入跨区域成长阶段。

### 内院与陨落心炎争夺战
- entry_type: battle
- aliases: 天焚炼气塔危机
- related_people: 萧炎|韩枫|苏千|紫妍
- importance: core

参战方：迦南学院内院阵营、韩枫等黑角域势力、陨落心炎失控因素。
胜负：学院虽付出巨大压力，但危机最终被控制，萧炎在其中获得巨大收益。
代价：内院秩序受到强烈冲击，黑角域与学院冲突全面升级。
后果：萧炎拿到中期最关键的异火成长节点之一，且在年轻一辈中的地位大幅提升。

### 星陨阁与魂殿救药老战
- entry_type: battle
- aliases: 救师之战|魂殿大战
- related_people: 萧炎|药尘|风尊者|魂殿
- importance: core

参战方：以萧炎、风尊者等人为核心的一方 vs 魂殿体系。
胜负：萧炎阵营在巨大压力下完成关键目标，药老线由被动受困转入主动回归。
代价：萧炎与魂殿之间再无回旋空间，双方敌对上升到不死不休。
后果：主角第一次真正以“救老师、扛大旗”的身份站到大陆强者博弈前台。

### 净莲妖火争夺战
- entry_type: battle
- aliases: 妖火大战
- related_people: 萧炎|净莲妖圣|魂族|古族|雷族|炎族
- importance: core

参战方：多方斗圣级阵营围绕净莲妖火展开混战。
胜负：萧炎成为最大受益者之一，成功把妖火转化为自己后期成帝前的重要拼图。
代价：大量高端战力消耗，后期阵营关系进一步定型。
后果：萧炎在异火路线与斗圣层面双双完成跃迁，终局对决的筹码显著增加。

### 古帝洞府争夺战
- entry_type: battle
- aliases: 洞府之争|帝品雏丹争夺
- related_people: 萧炎|魂天帝|古元|烛坤
- importance: core

参战方：萧炎阵营联军 vs 魂族及相关顶级战力。
胜负：围绕古帝传承的争夺一度极其胶着，魂天帝抢先迈出关键一步，联军被迫转入拖延与保全模式。
代价：大陆层面的战略主动权短时间内落入魂族之手。
后果：萧炎必须用更极限的方式争夺时间与传承，否则整个大陆秩序都将被改写。

### 双帝之战
- entry_type: battle
- aliases: 炎帝战魂天帝|终局决战
- related_people: 萧炎|魂天帝
- importance: core

参战方：炎帝萧炎 vs 魂天帝。
胜负：萧炎最终获胜，终结魂族最高野心。
代价：大陆经历浩劫级动荡，原有顶级格局被重新洗牌。
后果：斗气大陆从“斗帝断层后的绝望时代”重新看到秩序修复与未来可能，萧炎则以炎帝身份完成全书的最后命题。

## 检索提示（二期补强）

### 推荐新增过滤字段
- entry_type: topic
- aliases: 二期检索字段
- importance: medium

在 v2 稿件中，建议额外利用以下 metadata 做精确召回：
1. `chapter_range`：适合按剧情阶段追问某段发生了什么。
2. `rank`：适合异火榜、战力榜、资源等级问答。
3. `battle_result`：后端如需增强，可在后处理阶段从战役正文中抽取。
4. `stance`：可用于势力阵营态度问答。
5. `expansion_phase`：用于区分 v1/v2 文档来源与回滚策略。

- aliases: v3规划
