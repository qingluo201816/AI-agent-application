# 文档 1：世界观总设定

## 1.1 作品基调
- **题材定位**：东方玄幻、成长、群像、宗门、王朝博弈、上古遗秘
- **整体氛围**：宏大、神秘、压抑中带希望
- **叙事风格**：主线升级 + 阴谋推进 + 人物宿命交织
- **核心主题**：
  - 人能否挣脱出身与天命
  - 力量与代价是否等价
  - 守护与掠夺之间的边界
  - 记忆、传承与文明轮回

## 1.2 世界名称与结构
- **世界主名**：玄溟界
- **世界结构**：一界三域九洲
  - **天衡域**：宗门林立，灵脉鼎盛，修行文明最发达
  - **赤荒域**：妖族、凶地、遗迹众多，资源丰富但危险极高
  - **幽墟域**：古战场、禁区与失落文明聚集地，常年被迷雾与空间裂隙笼罩
- **九洲概念**：九大洲对应不同修行传统、地理环境与政治势力，可作为后续副本与地图拓展核心

## 1.3 修炼体系

### 1.3.1 大境界
1. **炼体境**：淬炼血肉筋骨，打基础
2. **聚气境**：感知天地灵气，形成气海
3. **灵府境**：体内开辟灵府，可储术法与本命灵印
4. **化魂境**：神识初成，可御器、感知杀机、稳固道心
5. **命轮境**：凝聚命轮，寿元提升，开始真正走上强者之路
6. **通玄境**：感悟天地规则碎片，术法威能大幅增强
7. **归真境**：返璞归真，自身大道初成
8. **圣境**：可镇一域，开宗立派，改天换地
9. **帝境**：传说境界，几近绝迹，仅存于上古记载

### 1.3.2 修炼资源
- 灵石：基础流通资源
- 灵药：炼丹与疗伤的重要材料
- 妖核：妖兽力量结晶，可用于炼器、阵法、突破
- 遗骨/古碑：上古传承载体
- 灵脉：宗门与王朝争夺核心
- 命火晶：涉及命轮与寿元的稀有资源

### 1.3.3 力量规则
- 力量越强，越需要承受心魔、天劫、因果反噬
- 天赋决定起点，心性决定上限，机缘决定速度
- 术法并非越强越好，契合自身大道才可真正发挥威能
- 越接近世界本源，越容易触碰被隐藏的真相与禁忌

## 1.4 世界核心矛盾
- 表面矛盾：宗门、王朝、世家、妖族之间的资源与统治权争夺
- 深层矛盾：上古时代的“封天之战”并未真正结束，整个玄溟界被某种古老意志封锁
- 终极矛盾：众生修行飞升所追求的“天外之境”，可能只是更高层囚笼

## 1.5 历史真相（供隐藏设定使用）
- 上古时代，玄溟界曾有完整飞升路
- 后来诸帝争道，引发“封天之战”，飞升路崩塌
- 世界本源被分裂为九块“界源碎印”，分别散落九洲
- 现今各大势力争夺的遗迹、秘境、圣物，其实都和界源碎印有关
- 主角命运将与其中一块碎印直接绑定

## 1.6 势力概览

### 正道宗门
- **太一玄宗**：天衡域第一宗门，底蕴深厚，重法统与秩序
- **青冥剑阁**：剑修圣地，门人孤高，战力极强
- **丹霞谷**：炼丹世家与医道圣地，表面中立，实则掌握大量情报

### 王朝势力
- **大曜皇朝**：控制天衡域中部大片疆土，皇族血脉神秘
- **北陵王庭**：尚武崇血统，与宗门关系微妙

### 异族与禁忌势力
- **万妖山**：妖族联盟，并非铁板一块
- **冥河教**：信奉古老黑暗意志，擅长献祭、夺魂、操控尸傀
- **无面楼**：遍布三域的杀手与情报组织，只认利益

## 1.7 世界禁忌
- 不可轻启上古祭坛
- 不可直视幽墟深处的“天裂之眼”
- 不可强行融合多种本源传承
- 不可在命轮未稳前触碰界源碎印

## 1.8 秘境系统
- **灵墟古境**：新手期重要机缘地，藏有失落传承
- **葬龙渊**：上古龙族陨落之地，危机与至宝并存
- **九曜天宫遗址**：牵引主线真相的重要遗迹
- **黑潮禁海**：偏群像与大事件推动场景
