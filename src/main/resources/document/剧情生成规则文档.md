# 文档 4：剧情生成规则文档

> 该文档用于约束 AI 在调用 RAG 时的生成方向，使输出更像“同一部作品”而不是散乱片段。

## 4.1 剧情生成原则
1. 每段剧情必须有明确目标：求生、试探、夺宝、破局、救人、复仇、揭密之一或其组合
2. 任何机缘都必须附带代价，不能无成本获得巨大收益
3. 人物行为必须符合既有性格与立场，不能为了推进剧情强行降智
4. 每次战斗至少推动一项内容：角色成长、关系变化、情报披露、阵营转变
5. 阶段性爽点后必须伴随更高层风险，维持张力

## 4.2 情节模板

### 模板 A：任务驱动型
- 输入：某人发布任务 / 宗门试炼 / 外出调查
- 常见流程：接令 → 途中异常 → 小冲突 → 线索反转 → 主冲突 → 留下后患
- 适用：支线推进、人物磨合、世界补充

### 模板 B：秘境探险型
- 输入：遗迹开启、地图现世、古碑共鸣
- 常见流程：进入 → 规则发现 → 队伍分裂/竞争 → 幻境或试炼 → 核心宝物争夺 → 离开后被追杀
- 适用：升级、揭露历史、制造宿敌

### 模板 C：阴谋揭露型
- 输入：城池异象、人口失踪、阵法波动、密信截获
- 常见流程：调查 → 怀疑对象 → 发现更深层黑手 → 营救/阻止失败或半成功 → 真相只揭开一角
- 适用：主线推进、反派塑造、世界阴影增强

### 模板 D：关系升温型
- 输入：同行、受伤、共患难、秘密交换
- 常见流程：互相提防 → 被迫合作 → 某人展露脆弱面 → 信任提升 → 因外部事件再度压抑情绪
- 适用：感情线、搭档线、同伴线

## 4.3 文风约束
- 避免现代口语感过强的表达
- 场景描写强调压迫感、空间层次、灵气流动、危险征兆
- 人物对话应体现身份差异：宗门弟子、散修、皇族、妖族说话方式不同
- 战斗描写兼顾动作、感知、心理判断与术法效果
- 真相披露遵循“七分隐藏，三分显露”原则

## 4.4 检索标签建议
- 人物类：主角、女主、宿敌、师父、反派、妖族、皇族
- 设定类：修炼体系、宗门、王朝、妖族、秘境、禁忌、界源碎印
- 事件类：灭门案、封天之战、灵墟古境、黑潮南侵、祭天案
- 情感类：信任、试探、背叛、守护、宿命、合作、牺牲
- 场景类：山门、古城、秘境、遗迹、禁区、边境、王都
